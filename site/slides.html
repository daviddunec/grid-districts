The problem

Discretion is the vulnerability

  • District lines are drawn by people whose careers depend on where they fall.
  • Rucho v. Common Cause (2019): federal courts cannot police partisan gerrymandering — the fix must come from the political branches.
  • Every reform debate stalls on the same question: who can be trusted to draw the lines?
The proposal

Remove the discretion from the benchmark

An algorithm that sees only one thing: how many people live in each square mile.
Never race. Never party. Never an address. Never an incumbent.
  • Deterministic: the same input produces the identical map on any computer, run by anyone.
  • Open source: no one controls it; everyone can check it.
  • Proposed as a published baseline, not a mandated map: beat the baseline or explain why.

Verification, not trust

Every claim is checkable twice

  • Every state's population verified against the official Census tables and the exact sum of its own census blocks.
  • An independent verification suite — written against a published data contract, forbidden from reading the engine's code — re-derives every result.
  • The development defect log ships with the code: eleven issues, each with root cause, fix, and prevention rule. A system claiming verifiability must show its own errors.
Since 1950

The process doesn't care who is in power

  • The same engine, applied to all eight apportionment cycles, 1950 → 2020, using each decade's actual seats and county census counts.
  • Maps evolve smoothly with population — never lurching with political control, because there is no political control anywhere in the loop.
  • Interactive: every state, every decade, on the accompanying website.
Historical maps are labeled approximations (block-level data predates digital records); they demonstrate the process, not reconstructed geography.
The hard question, answered honestly

The Voting Rights Act

  • A demographics-blind algorithm cannot, by itself, guarantee VRA-required districts.
  • That is why this is a baseline, not a binding map: states keep drawing; existing law — with an explicit VRA savings clause — keeps governing.
  • The baseline supplies what Rucho said must come from legislatures: a neutral, public standard against which every enacted map is measured.
The mechanism

The FAIR Baseline Act (discussion draft)

  • The neutral baseline map for every state is computed and published each cycle from the open specification.
  • Every enacted congressional plan is published alongside the baseline with a standard metrics comparison.
  • Deviations are documented and justified in public. No federal takeover; a pure disclosure standard under the Elections Clause.
The ask

Three steps

  • 1. A hearing on algorithmic baseline standards for the post-2030 cycle.
  • 2. A technical review by GAO or the Census Bureau — the code and data are public today.
  • 3. Consideration of the baseline-disclosure standard in the FAIR Baseline Act discussion draft.
